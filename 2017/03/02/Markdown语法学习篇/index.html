Markdown标题语法

Markdown支持两种标题书写语法：Setext形式和Atx形式

Setext形式

=最高阶标题和-第二阶标题。

Markdown中书写形式如下：

A First Level Header
====================
A Second Level Header
---------------------

Atx形式

根据#来标记是否为标题及标题的等级。

Markdown中书写形式如下：

# A First Level Header
## A Second Level Header
### A Third Level Header
#### A Fourth Level Header
##### A Fifth Level Header
###### A Sixth Level Header

注意：#与标题首字间需空一格。
